# Inventory Reports Tab

## Purpose
Summarizes glove and sleeve inventory by class and status for reporting and analysis.

## Columns
- **A: Item Type** - Glove or Sleeve.
- **B: Class** - Item class (0, 2, 3, etc.).
- **C: Status** - Inventory status (Assigned, On Shelf, etc.).
- **D: Count** - Number of items in this category.

## Interactions
- Aggregates data from Gloves and Sleeves tabs.
- Used for purchase planning and inventory review.

## Example
| Item Type | Class | Status   | Count |
|-----------|-------|----------|-------|
| Glove     | 2     | Assigned | 50    |
| Sleeve    | L     | On Shelf | 20    |

## Cross-Tab Dependencies
- Gloves and Sleeves tabs for source data.
