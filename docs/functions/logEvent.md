# logEvent

## Purpose
Logs events and messages for debugging and auditing purposes.

## How it Works
- Receives a message and optional log level.
- Writes the log entry to a designated log tab or output.

## Example Usage
Called throughout the code to record significant actions or errors.

## Inputs
- message: String
- level: String (default 'INFO')

## Outputs
Log entry written.

## Side Effects
None.
