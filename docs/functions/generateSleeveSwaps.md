# generateSleeveSwaps

## Purpose
Generates the Sleeve Swaps tab, including workflow columns and user interaction columns for managing sleeve swaps.

## How it Works
- Reads Employees and Sleeves data.
- Builds swap rows for each employee and sleeve class.
- Sets up visible and hidden workflow columns.

## Example Usage
Called during sheet initialization or when swap data needs to be refreshed.

## Inputs
None.

## Outputs
Sleeve Swaps tab populated and formatted.

## Side Effects
May overwrite existing swap data.
