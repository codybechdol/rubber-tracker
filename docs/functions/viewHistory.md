# viewHistory

## Purpose
Displays the history of changes and actions taken in the spreadsheet for auditing and review.

## How it Works
- Retrieves log entries and change history.
- Displays them in the History tab or a custom UI.

## Example Usage
Called from the custom menu or during audits.

## Inputs
None.

## Outputs
History tab or UI updated with change log.

## Side Effects
None.
