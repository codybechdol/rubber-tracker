# buildSheets

## Purpose
Builds and initializes all required tabs in the spreadsheet, including headers and initial data.

## How it Works
- Creates or resets tabs for Gloves, Sleeves, Employees, Glove Swaps, Sleeve Swaps, Inventory Reports, and History.
- Sets up headers and initial formatting.

## Example Usage
Called from the custom menu to reset or initialize the spreadsheet structure.

## Inputs
None.

## Outputs
Tabs created and initialized.

## Side Effects
May overwrite existing data if tabs are reset.
