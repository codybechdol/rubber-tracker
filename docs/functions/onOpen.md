# onOpen

## Purpose
Initializes the spreadsheet UI and menu when the document is opened.

## How it Works
- Adds custom menu items for building sheets, viewing history, running reclaims check, and updating inventory reports.
- Ensures users have access to key functions from the UI.

## Example Usage
Automatically invoked when the spreadsheet is opened.

## Inputs
None.

## Outputs
Custom menu added to the spreadsheet UI.

## Side Effects
None.
