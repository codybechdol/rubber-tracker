# generateGloveSwaps

## Purpose
Generates the Glove Swaps tab, including workflow columns and user interaction columns for managing glove swaps.

## How it Works
- Reads Employees and Gloves data.
- Builds swap rows for each employee and glove class.
- Sets up visible and hidden workflow columns.

## Example Usage
Called during sheet initialization or when swap data needs to be refreshed.

## Inputs
None.

## Outputs
Glove Swaps tab populated and formatted.

## Side Effects
May overwrite existing swap data.
